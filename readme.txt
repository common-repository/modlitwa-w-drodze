== Description ==

"Modlitwa w drodze" - http://modlitwawdrodze.pl/

Możesz na swojej stronie wykorzystywać widget, który prezentuje adres do Ewangelii na dzień umożliwia odtworzenie rozważań na dany dzień.

== Installation ==

1. Wgraj katalog `modlitwa-w-drodze` do katalogu `/wp-content/plugins/`
1. Aktywuj wtyczkę w zakładce 'Wtyczki' w Wordpressie
1. Aktywuj widget w `Wygląd` > `Widgety`
